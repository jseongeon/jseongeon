## A new way to connect **Business · People · Data**

단순히 기획서를 작성하는 사람이 아니라  
**전략 수립 → 실행 → 데이터 분석 → 구조 개선**까지  
하나의 사이클을 스스로 설계하고 운영합니다.

- 비즈니스 KPI 구조를 새로 디자인하고  
- 조직과 사람의 동기부여 시스템을 만들고  
- 데이터를 통해 “왜 이런 결과가 나왔는지” 설명합니다.
